# QuickDrop

QuickDrop is an improved version of NearDrop, a QuickShare client for macOS. QuickDrop includes apps for macOS, iOS, iPadOS, and Android; better error handling; and additional features such as auto-start and auto-accept files.

## Downloads

- QuickDrop for Mac: [Download from GitHub Releases](https://github.com/leonboe1/QuickDrop/releases) or download it from the [Mac App Store](https://apps.apple.com/de/app/quickdrop-android-transfer/id6740147178) to support this project
- QuickDrop iOS and iPadOS: [Download on the App Store](https://apps.apple.com/de/app/quickdrop-android-transfer/id6740147178)
- QuickDrop for Android: [Download on Google Play](https://play.google.com/store/apps/details?id=com.leonboettger.quickdrop)

## Compile from source
Download and open this repository in Xcode. 

You have to re-sign the project with your own Apple ID. 

Also, keep in mind that you may need to replace the prefix of the Bundle Identifiers with your identifier.
Those can be found in Project -> Targets -> Signing -> Bundle Identifier

Please also change the App Group names accordingly. This can be done by editing the .entitlements files in the root folder of this repository.
